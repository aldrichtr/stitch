# Find-InvokeBuildTaskFile

## SYNOPSIS
Find all "task type" files. 
These are files that contain "extensions" to the task types. 
They define a
function that creates tasks.

## SYNTAX

```
Find-InvokeBuildTaskFile [[-Path] <String[]>] [<CommonParameters>]
```
